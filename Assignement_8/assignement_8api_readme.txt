Setup Instructions:
1. Install Required Libraries:
bash
    pip install openai speechrecognition

2. Set OpenAI API Key:
bash
    export OPENAI_API_KEY='your-api-key-here'

3. Run the program:
bash
    python3 assignement_8api.py

Key Features:
1. Uses English
2. Image Generation:
    - Uses OpenAI's DALL-E API
    - Generates 512x512 images
3. Maintained speech capture reliability (3-second pause treshold)
4. Show the actual image URL when generation is complete
5. User Guidance:
    - Step-by-step instructions
    - Clear feedback at each stage
Note: Each image generation costs credits on your OpenAI account.
